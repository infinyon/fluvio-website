Manage and view SPU Groups (SPGs)

SPGs are groups of SPUs in a cluster which are managed together.

Usage: fluvio cluster spg <COMMAND>

Commands:
  create  Create a new managed SPU Group
  delete  Delete a managed SPU Group
  list    List all SPU Groups

Options:
  -h, --help
          Print help (see a summary with '-h')
